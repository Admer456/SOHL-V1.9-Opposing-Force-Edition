I have several sprites in my folder, how can I switch between them, not opening new SE windows? [top]
you can use buttons Back(Q) and Next(W) to load prev/next sprite in folder or pak from which current sprite was loaded. If you want to skip to some another sprite, you can select it from drop-down menu on main toolbar.

 
How can I load sprite from PAK file? [top]
click open PAK File from the File menu. In the window that pops up, find and open the folder in which the
sprites are found, and select a sprite from the list.

 
How can I switch between sprite files from pak, not calling exploration window each time? [top]
Simply select a sprite from the drop-down menu in the Sprite Explorer window.

 
How can I decompile existing sprite? [top]
There are many ways to decompile sprites.
You can save single frames to .bmp's or .tga's. Just select the frame you want, and press an icon.
You can also save multiple frames to .bmp's or .gif's. Select all the frames you want to save and press an icon.
Lastly, you can decompile the whole sprite to a series of .bmp files, and a .qc file, or as an animated .gif
file.

 
I want to associate SE with .spr extension in my system. What to do? [top]
Select Configuration from the edit menu in SE, and click register .spr files with Sprite Explorer.

 
I want to clear recent sprites/paks list, how can I do that? [top]
Go to Configurations from the Edit menu, and click clear recent files list.

 
What is [FAT]/[PAK] text in right bottom corner? [top]
[FAT] Is displayed if you are veiwing a sprite from a folder, and [PAK] is shown if the sprite is in a PAK
file.

 
How can I view sprite fitted to the preview rect? [top]
Click the Fit preview area icon in the preveiw area.

 
When sprite file is added to recent list? [top]
After you have opened a sprite from a folder or selected it from drop-down menu and exited SE, it will be listed when you return.

 
I think I found the bug, how can I tell you about that? [top]
Send an email to slava@wrenchsoftware.com, describing what you have found, and it should be fixed for the next
release.
